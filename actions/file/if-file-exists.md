# If File Exists

## Description
Marks the beginning of a conditional block of actions depending on whether a file exists or not.

## Syntax
```
IF (File.IfFile.Exists File: `[file path]`) THEN
```

## Input Parameters

| Parameter | Required | Accepts | Default Value | Description |
|-----------|----------|---------|---------------|-------------|
| If file | Yes | Exists, Doesn't exist | Exists | The state of the file to check |
| File path | Yes | File | - | The full path to look for the file |

## Output Variables
This action doesn't produce any variables.

## Exceptions
This action doesn't include any exceptions.
