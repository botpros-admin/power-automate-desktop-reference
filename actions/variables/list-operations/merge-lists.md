# Merge Lists

## Description
Merge two lists into one.

## Syntax
```
Variables.MergeLists FirstList: `[first list]` SecondList: `[second list]` OutputList=> OutputList
```

## Input Parameters

| Parameter | Required | Accepts | Default Value | Description |
|-----------|----------|---------|---------------|-------------|
| First list | Yes | List of General values | - | First list to merge |
| Second list | Yes | List of General values | - | Second list to merge |

## Output Variables

| Variable | Type | Description |
|----------|------|-------------|
| OutputList | List of General values | The merged list. Original lists unchanged |

## Exceptions

| Exception | Description |
|-----------|-------------|
| The lists supplied are of incompatible types | Lists have different element types |
