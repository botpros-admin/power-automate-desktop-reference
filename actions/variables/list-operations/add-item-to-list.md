# Add Item to List

## Description
Append a new item to a list.

## Syntax
```
Variables.AddItemToList Item: `[item]` List: `[list]`
```

## Input Parameters

| Parameter | Required | Accepts | Default Value | Description |
|-----------|----------|---------|---------------|-------------|
| Add item | Yes | General value | - | A value or variable to add to the list. Can provide a list of values to append multiple elements |
| Into list | Yes | List of General values | - | A list variable to append the new elements to |

## Output Variables

| Variable | Type | Description |
|----------|------|-------------|
| NewList | List of General values | The new list |

## Exceptions
This action doesn't include any exceptions.
