# Sort List

## Description
Sort the items of a list. Use items of the same type.

## Syntax
```
Variables.SortList.SortList List: `[list]`
```

## Input Parameters

| Parameter | Required | Accepts | Default Value | Description |
|-----------|----------|---------|---------------|-------------|
| List to sort | Yes | List of General values | - | The list to sort |
| Sort by list item's properties | No | Boolean value | False | Enable to sort items by specific property |
| First property to sort by | Yes (if sorting by properties) | Text value | - | Name of item's property to sort by |
| Sort | No | Ascending, Descending | Ascending | Sort direction for first property |
| Second property to sort by | No | Text value | - | Second property name |
| Sort | No | Ascending, Descending | Ascending | Sort direction for second property |
| Third property to sort by | No | Text value | - | Third property name |
| Sort | No | Ascending, Descending | Ascending | Sort direction for third property |

## Output Variables
This action doesn't produce any variables.

## Exceptions
This action doesn't include any exceptions.
