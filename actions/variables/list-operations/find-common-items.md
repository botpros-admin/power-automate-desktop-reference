# Find Common List Items

## Description
Compare two lists and create a new list with the items that are common to both.

## Syntax
```
Variables.FindCommonListItems FirstList: `[first list]` SecondList: `[second list]` OutputList=> IntersectionList
```

## Input Parameters

| Parameter | Required | Accepts | Default Value | Description |
|-----------|----------|---------|---------------|-------------|
| First list | Yes | List of General values | - | First list to compare |
| Second list | Yes | List of General values | - | Second list to compare |

## Output Variables

| Variable | Type | Description |
|----------|------|-------------|
| IntersectionList | List of General values | New list of common items |

## Exceptions
This action doesn't include any exceptions.
