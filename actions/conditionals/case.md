# Case

## Description
An expression that, if met, a block of actions associated with that particular case runs.

## Syntax
```
CASE = `[value]`
    [actions]
```

## Input Parameters

| Parameter | Required | Accepts | Default Value | Description |
|-----------|----------|---------|---------------|-------------|
| Operator | Yes | Equal to (=), Not equal to (<>), Greater than (>), Greater than or equal to (>=), Less than (<), Less than or equal to (<=), Contains, Does not contain, Is empty, Is not empty, Starts with, Does not start with, Ends with, Does not end with, Is blank, Is not blank | Equal to (=) | The comparison operator of this case |
| Value to compare | Yes | * | - | Enter a value to compare with the switch-block value |

## Output Variables
This action doesn't produce any variables.

## Exceptions
This action doesn't include any exceptions.
