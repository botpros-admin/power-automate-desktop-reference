# Conditional Actions

This section contains all actions related to conditional logic and flow control in Power Automate Desktop.

## Available Actions

1. [If](./if.md) - Basic conditional execution
2. [Else If](./else-if.md) - Secondary conditions
3. [Else](./else.md) - Default execution path
4. [Switch](./switch.md) - Multi-way branching
5. [Case](./case.md) - Switch condition
6. [Default Case](./default-case.md) - Default switch path

## Common Use Cases
- Branching logic based on conditions
- Multiple condition checking
- Pattern matching
- Value-based execution paths
