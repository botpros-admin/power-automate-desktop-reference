# Loop Actions

This section contains all actions related to creating and controlling loops in Power Automate Desktop.

## Available Actions

1. [Exit Loop](./exit-loop.md)
2. [For Each](./for-each.md)
3. [Loop](./loop.md)
4. [Loop Condition](./loop-condition.md)
5. [Next Loop](./next-loop.md)

## Common Use Cases
- Iterate through lists and collections
- Process data tables row by row
- Perform actions a specific number of times
- Execute actions while conditions are met
- Control loop flow with exit and continue actions
