# Close Outlook

## Description
Close a previously launched Outlook instance.

## Syntax
```
Outlook.Close Instance: `[instance]`
```

## Input Parameters

| Parameter | Required | Accepts | Default Value | Description |
|-----------|----------|---------|---------------|-------------|
| Outlook instance | Yes | Outlook instance | - | The Outlook instance to close |

## Output Variables
This action doesn't produce any variables.

## Exceptions

| Exception | Description |
|-----------|-------------|
| Failed to close Outlook instance | Problem closing the instance |
