# If Folder Exists

## Description
Mark the beginning of a conditional block of actions depending on whether a folder exists or not.

## Syntax
```
IF (Folder.IfFolderExists.Exists Path: `[folder path]`) THEN
```

## Input Parameters

| Parameter | Required | Accepts | Default Value | Description |
|-----------|----------|---------|---------------|-------------|
| If folder | Yes | Exists, Doesn't exist | Exists | Choose the state of the folder to check |
| Folder path | Yes | Folder | - | Enter or choose the full path of the folder, or a variable containing the folder, to check its state |

## Output Variables
This action doesn't produce any variables.

## Exceptions
This action doesn't include any exceptions.
