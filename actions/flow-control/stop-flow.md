# Stop Flow

## Description
Terminates the flow.

## Syntax
```
EXIT Code: [exit code]
```

## Input Parameters

| Parameter | Required | Accepts | Default Value | Description |
|-----------|----------|---------|---------------|-------------|
| End flow | Yes | Successfully, With error message | Successfully | Terminate the execution of the flow, either successfully or with an error |
| Error message | Yes (if ending with error) | Text value | - | The error message to return to the flow caller upon exit |

## Output Variables
This action doesn't produce any variables.

## Exceptions
This action doesn't include any exceptions.
