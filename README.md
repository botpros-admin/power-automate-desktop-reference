## Structure

Each category folder contains:

- `README.md` with category overview and action list
- Individual markdown files for each action containing:
  - Description
  - Input parameters
  - Output variables
  - Exceptions
  - Exact copy/paste syntax
  - Usage examples

## Usage

1. Browse the category index above
2. Click through to specific actions
3. Copy the exact syntax provided
4. Use the syntax in your PAD flows
