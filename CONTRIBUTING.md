# Contributing to Power Automate Desktop Reference

## How to Contribute

### Adding New Actions
1. Follow the established format in the respective category folder
2. Include full documentation:
   - Description
   - Exact syntax
   - All parameters with types and descriptions
   - Output variables
   - Exceptions
   - Example usage
   - Best practices

### Documentation Format
Each action should be documented using this template:

```markdown
# Action Name

## Description
Clear description of what the action does

## Syntax
```
Exact.Copy.Syntax Parameter: `` Output=> Variable
```

## Parameters
| Parameter | Required | Accepts | Default | Description |
|-----------|----------|---------|---------|-------------|

## Output Variables
| Variable | Type | Description |
|----------|------|-------------|

## Exceptions
| Exception | Description |
|-----------|-------------|

## Example Usage
```powershell
# Example code
```

## Best Practices
1. Practice 1
2. Practice 2
```

### Pull Request Process
1. Create feature branch from main
2. Follow documentation format
3. Test all example code
4. Update relevant indexes
5. Submit PR with clear description

### Style Guide
- Use clear, concise language
- Include practical examples
- Document all parameters
- Include error handling
- Link to related actions
